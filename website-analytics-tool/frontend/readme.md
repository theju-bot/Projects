# Frontend

React application with TypeScript for the Website Analytics Tool dashboard.

**Author:** Thesigan  
**Repository:** [github.com/theju-bot/Projects](https://github.com/theju-bot/Projects) (this app lives in the `website-analytics-tool/` directory)

## Features

- Modern React 19 with TypeScript
- Analytics dashboard with charts
- Site management interface
- User authentication (login/register)
- Real-time data with React Query
- Responsive design with Tailwind CSS
- shadcn/ui components

## Tech Stack

- React 19 with TypeScript
- Vite for build tooling
- Redux Toolkit for state management
- TanStack React Query for data fetching
- React Router DOM for routing
- Tailwind CSS v4 for styling
- shadcn/ui components (Radix UI)
- Recharts for data visualization
- React Hook Form with Zod validation
- Axios for HTTP requests

## Installation

```bash
npm install
```

## Environment Variables

```env
VITE_API_URL=http://localhost:5000/api
```

## Running

```bash
npm run dev
```

## Building

```bash
npm run build
npm run preview
```
